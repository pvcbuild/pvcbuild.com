Plugins for People

Star on GitHub Tweet it!

When I first heard about watsi, I saw a new future for crowdfunded charities. I've been thinking about how I can help constantly.

Another cause that I care a great deal about -- Open Source Software -- also needs help. Particularly in the .NET community.

In an effort to engage people in our community to both expand it and help others get needed medical care, I'm personally funding Round 1 of Plugins for People.

For the first 50 PVC plugins built and released under MIT, BSD or the Apache 2 license we will donate $50 to a watsi patient on behalf of the author.

Fine Print: To be included in this round, the plugin must be considered generally useful by the community at large. To this end we have created a number of Plugin Request issues on GitHub. If your idea is not tracked here, submit a new request and lets talk about it!

To get involved, comment on the issue for the plugin you're working on and follow these instructions:

1. Install the PVC command line interface via Chocolatey.
cinst pvc
2. Use the PVC init scripts
to create a new plugin solution
pvc init plugin
3. Build something great
and help a person in need!
